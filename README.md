# Deskripsi
Project ini merupakan frontend untuk Sistem Peminjaman Ruangan Kampus yang dikembangkan menggunakan React dengan TypeScript.
Frontend ini terhubung dengan backend ASP.NET Core untuk menampilkan dan mengelola data ruangan.

## Fitur
- Menampilkan daftar ruangan
- Menambahkan data ruangan
- Mengedit data ruangan
- Menghapus data ruangan
- Terhubung ke REST API backend

### Teknologi yang Digunakan
- React
- TypeScript
- Fetch API
- Node.js
- Git & GitHub
